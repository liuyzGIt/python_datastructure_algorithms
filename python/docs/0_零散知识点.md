# 1 求余函数的实现原理
## 1.1 python求余操作
`a = b % c ` 等价于 `a = b - c * (b // c)`  
a为余数，b为除数，c为被除数

-1 % 4 = -1 - 4 * (-1 // 4) = -1 + 4 = 3  
-1 // 4 = -1

## 1.2 整数(//)保留整数的下界
1 / 4 = 0.25 所以 1 // 4 = 0  
-1 // 4 = -0.25 所以 -1 // 4 = -1
